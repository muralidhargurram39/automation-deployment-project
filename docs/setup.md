# Setup Guide

Prerequisites

- Docker
- Docker Compose
- Git
- Java 17
- Maven

Containers

- Jenkins
- SonarQube
- PostgreSQL
- Nexus
- Tomcat

Build

mvn clean package

Run

docker compose up -d

Jenkins

http://localhost:8080

SonarQube

http://localhost:9000

Nexus

http://localhost:8082

Tomcat

http://localhost:8081
