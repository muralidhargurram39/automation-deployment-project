# Enterprise DevSecOps CI/CD Pipeline

## Overview

This project demonstrates an end-to-end Enterprise DevSecOps pipeline using modern DevOps tools.

## Tech Stack

- Java 17
- Maven
- Jenkins
- SonarQube
- Nexus Repository
- Docker
- Trivy
- Tomcat
- Kubernetes (Upcoming)
- Helm (Upcoming)
- Argo CD (Upcoming)
- Prometheus (Upcoming)
- Grafana (Upcoming)
